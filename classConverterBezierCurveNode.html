Detailed Description

ConverterBezierCurveNode

Node that displays a complex slope and curve editing widget used for adjusting values as a function of f(x).

Converts an intensity map value x into a new intensity value x' with a function f(x) that is defined by a series of curves and slopes created in the widget. The x value is calculated via finding the vertical line intercept on the line segments of the slopes.
